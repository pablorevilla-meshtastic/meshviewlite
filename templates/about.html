About MeshviewLite

Lightweight Meshtastic MQTT collector + web viewer.

What It Does

  • Collects MQTT packets and stores them in SQLite.
  • Maintains latest node state from nodeinfo, telemetry, and position packets.
  • Provides a web UI for node list, node detail, and map views.

API Endpoints

  • /api/nodes
  • /api/packets
